# d9servo 🚀

**d9servo** — это ультра-легкая и простая библиотека для управления сервоприводами на Arduino. Она создана для тех, кому стандартные функции кажутся запутанными, и кто хочет управлять моторами одной понятной командой.

## ✨ Особенности
- **Легкость:** Минимальный расход памяти.
- **Два режима:** Отдельные методы для обычных серво (180°) и моторов постоянного вращения (360°).
- **Понятный синтаксис:** Забудь про сложные расчеты, используй только нужные значения.

## 📦 Установка
1. Скачай этот репозиторий как `.zip`.
2. В Arduino IDE выбери: `Скетч` -> `Подключить библиотеку` -> `Добавить .zip библиотеку...`.
3. Или просто распакуй папку в твою директорию `libraries`.
