## 📖 Обзор
Стандартное управление сервоприводами часто требует лишнего кода. **d9servo** создана, чтобы сделать ваши проекты чище. Она одинаково хорошо подходит как для обычных серво (180°), так и для моторов постоянного вращения (360°).

## ✨ Основные возможности
- **Два режима:** Отдельные методы `.rot()` для углов и `.speed()` для скорости.
- **Авто-масштабирование:** Больше не нужно использовать функцию `map()`.
- **Безопасность:** Встроенные ограничения диапазонов (0...180 и -100...100).
- **Энергосбережение:** Метод `.detach()` для отключения сигнала, чтобы серво не «гудела».

## 🛠 Схема подключения
- **Оранжевый/Желтый:** Сигнал (на цифровой пин, например D9)
- **Красный:** Питание (5V)
- **Коричневый/Черный:** Земля (GND)



## 🚀 Пример использования
```cpp
#include <d9servo.h>

d9servo myservo(9); // Серво на 9 пине

void setup() {
  myservo.init();
}

void loop() {
  myservo.rot(90);    // Поворот на 90 градусов
  delay(1000);
  myservo.speed(100); // Полный ход вперед (для 360°)
  delay(1000);
}
