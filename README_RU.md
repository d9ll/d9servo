

# d9servo 🚀
**Легкая и интуитивно понятная библиотека Arduino для удобного управления сервоприводами.**

[Read this in English (Читать на английском)](README.md)

## 📖 Обзор
Стандартное управление сервоприводами в Arduino часто требует ручного масштабирования значений и постоянных вызовов `write()`. **d9servo** упрощает этот процесс, предоставляя специальные методы для разных типов сервоприводов, делая ваш код чище.

## ✨ Основные возможности
- **Упрощенный интерфейс:** Отдельные методы для серво 180° и 360°.
- **Авто-масштабирование:** Больше не нужно использовать функцию `map()` в основном скетче для управления скоростью.
- **Безопасность:** Встроенные ограничения для предотвращения выхода за границы допустимых значений.
- **Энергосбережение:** Удобный метод `detach()` для устранения "гула" и экономии заряда.

## 🛠 Установка
1. Скачайте репозиторий в формате `.zip`.
2. В Arduino IDE перейдите в `Скетч` -> `Подключить библиотеку` -> `Добавить .ZIP библиотеку...`.
3. Выберите скачанный файл.

## 🚀 Примеры использования

### Обычная серво 180°
Управление точным углом с помощью метода `.rot()`.
```cpp
#include <d9servo.h>

d9servo myservo(9); // Инициализация на 9 пине

void setup() {
  myservo.init();
}

void loop() {
  myservo.rot(45);  // Поворот на 45 градусов
  delay(1000);
  myservo.rot(135); // Поворот на 135 градусов
  delay(1000);
}
