## 📖 Обзор
Стандартное управление сервоприводами в Arduino часто требует лишних действий: масштабирования значений через `map()` и постоянного контроля углов. **d9servo** решает эту проблему, разделяя логику управления для обычных серво и моторов постоянного вращения. Ваш код становится чище, короче и понятнее.

## ✨ Основные возможности
- **Упрощенный интерфейс:** Специальные методы `.rot()` для углов и `.speed()` для скорости.
- **Авто-масштабирование:** Больше не нужно вручную высчитывать значения для 360° серво.
- **Безопасность:** Библиотека сама ограничивает значения (0–180 для углов, -100...100 для скорости).
- **Тишина и экономия:** Метод `.detach()` позволяет мгновенно отключить сигнал, чтобы серво не «гудела».

## 🛠 Схема подключения
Обычно сервоприводы имеют три провода:
* **Коричневый/Черный:** GND (Земля)
* **Красный:** 5V (Питание)
* **Оранжевый/Желтый:** Сигнал (Сигнальный пин, например D9)



## 🚀 Примеры использования

### Обычная серво (180°)
```cpp
#include <d9servo.h>

d9servo myservo(9); // Подключаем к 9 пину

void setup() {
  myservo.init(); // Инициализация
}

void loop() {
  myservo.rot(0);   // Поворот в 0°
  delay(1000);
  myservo.rot(180); // Поворот в 180°
  delay(1000);
}
