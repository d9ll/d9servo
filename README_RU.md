## 📖 Обзор
Стандартное управление сервоприводами в Arduino часто требует ручного масштабирования значений и постоянных вызовов `write()`. **d9servo** упрощает этот процесс, предоставляя специальные методы для разных типов сервоприводов, делая ваш код чище и читабельнее.

## ✨ Основные возможности
- **Упрощенный интерфейс:** Специальные методы для серво 180° и 360°.
- **Авто-масштабирование:** Больше не нужно использовать функцию `map()` в основном скетче для управления скоростью.
- **Безопасность:** Встроенные ограничения для предотвращения выхода за границы допустимых значений.
- **Энергосбережение:** Удобный метод `detach()` для устранения "гула" и экономии заряда батареи.

## 🛠 Установка
1. Скачайте репозиторий в формате `.zip`.
2. В Arduino IDE перейдите в `Скетч` -> `Подключить библиотеку` -> `Добавить .ZIP библиотеку...`.
3. Выберите скачанный файл, и всё готово!

## 🚀 Примеры использования

### Обычная серво 180°
Управляйте точным углом с помощью метода `.rot()`.
```cpp
#include <d9servo.h>

d9servo myservo(9); // Инициализация на 9 пине

void setup() {
  myservo.init();
}

void loop() {
  myservo.rot(0);    // Минимальное положение
  delay(1000);
  myservo.rot(180);  // Максимальное положение
  delay(1000);
}
